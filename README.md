# Maltego-ATT&CK

Set of Transforms to query the MITRE ATT&CK database.

- Actor --> Tools, Techniques, Malware
- Tools --> Actor
- Tools --> Techniques
- Techniques --> Tools
- Techniques --> Actors
- Actor --> Malware (To-Do)
- Malware --> Actor (To-Do)
- Malware --> Techniques (To-Do)

### Prerequisites

- Python3 
- Maltego
- STIX2 Utilities Maltego Hub Item
- pyattck Python Library
